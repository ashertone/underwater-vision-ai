# 🌊 Underwater Image Enhancement System

## 📌 Overview
This project enhances underwater images affected by poor visibility, color distortion, and low contrast.

## 🚀 Features
- Upload underwater images
- Enhance clarity using image processing
- Adjust enhancement strength
- Download enhanced images

## 🧠 Technology Used
- Python
- OpenCV
- Streamlit

## 🎯 Application
Useful in maritime security, underwater exploration, and surveillance systems.
